## Run Locally

Clone the project

```bash
  git clone https://github.com/rkazi103/headphones-landing-page-react
```

Go to the project directory

```bash
  cd headphones-landing-page-react
```

Install dependencies

```bash
  npm install # NPM
  yarn # Yarn
```

Start the server

```bash
  npm start # NPM
  yarn start # Yarn
```
